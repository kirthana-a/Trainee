# 🌐 About Me Web Page

## 📌 Project Overview
This project is a simple **“About Me” web page** built using **HTML and CSS**. It showcases a personal profile with basic styling and layout.

---

## ✅ Features

- ✅ Complete HTML5 structure
- ✅ Personal profile card design
- ✅ Image with alt text (accessibility)
- ✅ Links to LinkedIn, GitHub, and Email
- ✅ Basic CSS styling (layout, colors, fonts)
- ✅ Responsive centered layout using Flexbox

---
