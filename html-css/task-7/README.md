# 📄 README – Cooking Hobby HTML Page

## ✅ Overview
This project is a simple HTML webpage about **my hobby: cooking**. It demonstrates how to use inline elements like `<span>` and `<a>` to improve readability, styling, and navigation within text.

---

## ✅ Features

### 1. Basic HTML Structure
The page uses proper HTML structure:
- `<!DOCTYPE html>`
- `<html>`, `<head>`, `<body>`
- Title and meta tags included

---

### 2. Paragraph Content
- Contains **2–3 paragraphs**
- Written in a **natural and human-friendly way**
- Describes cooking as a hobby

---

### 3. Inline Emphasis with `<span>`
Important words are wrapped in `<span>` elements with meaningful classes:

- `.keyword` → highlights important terms  
- `.highlight` → emphasizes ideas  
- `.important` → shows strong importance
