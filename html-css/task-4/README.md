# 🌐 About Me Web Page

## 📌 Project Description
This project is a simple **About Me web page** created using **HTML and CSS**.  
It demonstrates semantic HTML structure, proper content organization, and basic styling using an external CSS file.

---

## ✅ Features Implemented

### 🧱 HTML Structure
- Full HTML5 skeleton (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Meta tags:
  - Character encoding (`UTF-8`)
  - Responsive viewport
- Page title included

---

### 🏗️ Semantic Tags Used
- `<header>` – Page heading  
- `<nav>` – Navigation links  
- `<main>` – Main content  
- `<section>` – Content grouping  
- `<footer>` – Footer information  

---

### 📝 Content Organization
- Headings: `<h1>`, `<h2>`, `<h3>`
- Paragraph: `<p>` for description
- Lists: `<ul>` and `<li>` for skills and hobbies
