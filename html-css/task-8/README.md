# 📬 Contact Me Section

A simple and responsive **“Contact Me” section** built using HTML and CSS.  
This project demonstrates semantic HTML, a basic form, responsive images, and accessible form design.

---

## 🎯 Objective

The goal of this project is to:

- Practice semantic HTML structure  
- Build a clean contact form  
- Apply basic CSS styling  
- Ensure accessibility and responsiveness  

---

## 🚀 Features

- ✅ Semantic container using `<section>` and heading  
- ✅ Profile image with meaningful `alt` text  
- ✅ Contact form with:
  - Name input field  
  - Message textarea  
- ✅ Submit button  
- ✅ Responsive image that does not overflow  
- ✅ Clean layout with proper spacing  
- ✅ Accessible labels using `for` and `id`
