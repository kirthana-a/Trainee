Practice: Build a Simple “Contact Me” Section with an Image and a Basic Form
Create a small “Contact Me” section for a personal webpage that shows a profile image and a simple contact form. Focus on using semantic HTML where possible (e.g., <section>, <h2>) and basic CSS to make the image and form look neat and readable. Your form should collect at least a name and a message from the user, and be laid out clearly so it’s easy to understand and use.

 Add a semantic container (for example, a <section> with a heading) that introduces your “Contact Me” area, and insert a profile image using <img> with a meaningful alt attribute.
 Create a <form> inside this section with at least two labeled inputs (for example, a text input for “Name” and a <textarea> for “Message”), plus a submit button.
 Use basic CSS to: set a max width for the section, ensure the image scales nicely (e.g., doesn’t overflow on small screens), and add spacing between form fields so the layout is easy to read.
 Improve accessibility by explicitly associating each <label> with its corresponding input (using for and id), and ensure keyboard users can tab through the form in a logical order.
