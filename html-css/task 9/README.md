# 🎉 Community Events 2026 – Accessible Event Landing Page

## 📌 Project Overview
This project is a single-page Community Event landing website built using HTML and CSS, with a small amount of JavaScript for interactivity (modal popup). It highlights upcoming events and allows users to register easily.

The goal of this project is to demonstrate semantic HTML structure, accessibility, and modern layout design.

---

## 🚀 Features

- Semantic HTML structure (header, nav, section, footer)
- Internal navigation with anchor links
- Event cards with background images
- Responsive layout using modern CSS
- Event features list and schedule
- External link to Google Maps
- Registration modal with form inputs
- Accessible labels and form controls
