# 🍔 Recipe Page – HTML Lists Practice

## 📌 Overview
This project demonstrates how to create a simple **recipe webpage** using HTML with a focus on structured content. It uses:

- **Unordered lists (`<ul>`)** for ingredients  
- **Ordered lists (`<ol>`)** for step-by-step instructions  
- **Navigation links (`<nav>`)** for easy section access  

---

## 🎯 Objective
The goal of this exercise is to:
- Organize recipe content clearly
- Use semantic HTML elements for better readability
- Separate grouped data (ingredients) from sequential steps (directions)

---

## 🧱 HTML Elements Used

| Element | Purpose |
|--------|--------|
| `<header>` | Contains the navigation bar |
| `<nav>` | Provides links to sections |
| `<main>` | Wraps the main recipe content |
| `<section>` | Groups ingredients and steps |
| `<ul>` | Lists ingredients (unordered) |
| `<ol>` | Lists cooking steps in order |
| `<li>` | Represents each item in a list |
| `<footer>` | Displays copyright |

---

## 📝 Features Implemented
- ✅ Navigation menu with links to sections  
- ✅ Ingredients listed using `<ul>`  
- ✅ Step-by-step instructions using `<ol>`  
- ✅ Clean and structured layout  
- ✅ Simple, user-friendly recipe format  

---

## 💡 Why Use `<ul>` and `<ol>`?

- `<ul>` is used when **order doesn’t matter** (ingredients)  
- `<ol>` is used when **order is important** (steps)  

This improves:
- Readability  
- Accessibility  
- User experience
